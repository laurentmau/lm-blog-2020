## Comment je m'organise

Cette page décrit mon organisation : méthode, habitudes, process. Trois principes : habitudes, simplicité, numérique. Je la mets à jour quand quelque chose change. Le matériel et les logiciels sont sur [ma page /uses](/fr/uses.html).

### Les habitudes

Mon organisation repose sur des habitudes : automatiser les décisions répétitives pour garder la tête libre.

Les "fortes", tenues jour après jour : que mon agenda soit à jour (pro, perso), garder ma boîte mail vide, garder Todoist à jour et complet (toutes les tâches y sont). En période de stress ou de surcharge, ce sont elles qui lâchent en premier. C'est mon signal d'alerte.

Les "souples", créées pour tester ou progresser, avec une durée de vie limitée. Exemple : en 2018, "coder tous les jours" pendant un mois. Je code encore tous les jours.

### La simplicité

Ma méthode de base : "Zen to Done" de Leo Babauta, une version simplifiée de "Getting Things Done" (essayé plusieurs fois, jamais tenu). Adoptée en 2012. Quatorze ans après, elle tourne toujours.

Même filtre pour tout le reste : chaque fois que j'installe un process élaboré, je l'abandonne en quelques semaines. Donc je garde simple.

### Le numérique

Tout est numérique par défaut : mes informations me suivent, je travaille d'où je veux, sur la machine que je veux.

Une exception : les notes manuscrites. Écrire à la main m'aide à réfléchir, une Remarkable 2 me l'a fait découvrir. Je prends ces notes sur papier, puis je les repasse en numérique.

Les outils changent régulièrement : notes passées de Workflowy à Obsidian, tâches de Todoist à Trello puis retour à Todoist. Mes données restent à moi, les outils passent par-dessus. Dernier arrivé : un copilote IA, qui tient le contexte d'une dizaine de chantiers à ma place. Le détail est dans [mon post sur mon setup IA](/fr/posts/mon-setup-ia/).

### Les process : tâches, emails, agenda

**Tâches.** Une seule place : Todoist. Tout y va, même les micro-actions, pour garder l'esprit vide. Une revue hebdo complète. Chaque jour, trois choses importantes à accomplir : les MITs de Zen to Done. Ces trois-là, je les tiens.

**Emails.** Pas de tri automatique : tout arrive dans la boîte de réception, je passe chaque mail à la main. Rien à faire : j'archive. Moins de deux minutes : je le fais tout de suite. Plus long : ça part dans Todoist. Dix minutes par jour pour l'ensemble de mes comptes, boîtes vides.

**Agenda.** Chaque lundi matin, revue et nettoyage de la semaine : refuser les réunions évitables, protéger les créneaux libres, survoler la semaine suivante. Objectif : un agenda presque vide, qui se remplira d'imprévus utiles.

### Quinze ans après

Les outils ont tous changé au moins une fois. Les trois principes, non. L'IA n'a rien renversé : c'est le dernier outil que ce cadre a absorbé.
